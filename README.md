# weather-me
Command line tool to get the current weather of a given location.

## Usage
Valid arguments:
- zip code
- City State (e.g., Norwalk, OH)
- City Country (e.g., Paris, France)

### Example
#### Input
```
$ node app.js Dublin Ireland
```
